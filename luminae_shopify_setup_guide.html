Day 1 · Step 1
Store Setup & Environment
Create your Shopify Partner account, development store, and configure all basic store settings before touching any design.
⏱ Estimated time: 1 hour
🏗️ Shopify Partner Account
Create Shopify Partner Account
Use a fresh email address — not your personal one
  1. Go to partners.shopify.com
  2. Click Join for free
  3. Sign up with a new email (e.g. luminae.store@gmail.com)
  4. Complete the onboarding — select "I build stores for clients"
  5. Confirm your email address
Account email to use
luminae.store@gmail.com (create this fresh)
Create Development Store
Partners dashboard → Stores → Add store
  1. In Partners dashboard → click Stores in left sidebar
  2. Click Add store → Create development store
  3. Set store name: Luminae
  4. Select purpose: Create a store to test and build
  5. Click Save — your store URL will be luminae.myshopify.com
Development stores are free and don't require real payment details for testing orders.

Configure Shipping Zones
Settings → Shipping and delivery

Go to Settings → Shipping and delivery → Manage rates

  1. Delete any default rates
  2. Add shipping zone: India → select India as country
  3. Add rate: Standard Shipping — ₹79 — no conditions
  4. Add rate: Free Shipping — ₹0 — minimum order value ₹999
  5. Add rate: Express Shipping — ₹149 — no conditions
  6. Click Save
Set Up Taxes
Settings → Taxes and duties

Go to Settings → Taxes and duties

  1. Enable India as your tax region
  2. Set GST at 18% for body care and home products
  3. Check "Include tax in product prices" so displayed prices are final
  4. Save changes

Day 1 · Step 5
Apps Installation & Configuration
Install and configure all 6 required apps. Each one must be properly set up — not just installed.
⏱ Estimated time: 1.5 hours
🔌 Required Apps (all free)
1. Judge.me Product Reviews
Displays star ratings and written reviews on product pages. Add the 4 testimonials from the homepage copy as manual reviews.
  • Install from Shopify App Store → search "Judge.me"
  • Open Judge.me → Reviews → Import reviews → manually add the 4 testimonials
  • Settings → Review widget → set star color to #C2714F
  • Enable: Show star rating on collection pages
  • Enable: Review request emails (set delay to 7 days)
2. Shopify Search & Discovery
Powers the search bar and product filtering on collection pages.
  • Install from App Store → search "Shopify Search & Discovery"
  • Open app → Filters → Add filter: Product type, Scent, Size
  • Synonyms → add: "soap" = "bar soap", "oil" = "body oil"
  • Make sure search is enabled in Theme Customizer → Header
3. Shopify Bundles
Create the "Soap Trio" BOGO bundle directly within Shopify.
  • Install from App Store → search "Shopify Bundles"
  • Create bundle: The Soap Trio → select all 3 Soap Bar variants
  • Set bundle discount: Buy 3, save 30%
  • Add bundle to Ritual Bundles collection
4. Labeler — Product Labels
Adds "Bestseller", "New", and "Sale" badges on product images.
  • Install from App Store → search "Labeler Product Labels"
  • Create label: Bestseller — apply to Rose & Argan Oil, Soy Candle, Glow Ritual Kit
  • Create label: Gift Favourite — apply to both bundle products
  • Set label color to match brand: background #C2714F, text white
5. Shopify Subscriptions
Adds a "Subscribe & Save" option on repeat-purchase products like the soap and body oil.
  • Install from App Store → search "Shopify Subscriptions"
  • Create subscription plan: Subscribe & Save 15% — monthly delivery
  • Apply to: Soap Bar, Body Oil, Body Scrub
  • Enable widget on product pages
6. Shopify Marketplace Connect (optional)
Connects store to marketplaces. Install and configure basic settings to demonstrate it's set up.
  • Install from App Store → search "Shopify Marketplace Connect"
  • Connect to any marketplace (Amazon or Flipkart)
  • Map 2–3 products to show the integration is active
